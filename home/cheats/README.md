# My cheat sheet system

Kevin uses this to remember command line commands

## To generate a new cheat sheet for a command

- First update GENERATE-CHEAT.MD
- The run bash script generate-cheat.sh
- Then go into ~/.dotfiles
- run command bash copy.sh
- Check code into git
- Visit the other workstations and follow normal git process

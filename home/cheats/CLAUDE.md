# Cheats sheets

Used to produce a quick list of commands or what options exist for a command to remind me about how the command is used

Used by cheat command installed from 
https://github.com/cheat/cheat

Also review https://github.com/cheat/cheat/blob/master/CLAUDE.md

## Configuration

cheat is configured by ~/.config/cheat/conf.yml

## Cheat sheets locations

The cheat sheets it reads live in the directory locations below

There are two locations:

### These are cheat sheets I maintain
- /cheats

### These are community developed cheats
~/.config/cheat/cheatsheets

## A finished example of cheet sheet can be found at 

~/.config/cheat/cheatsheets/community/chown
